GEEK NOTES

This is a android app developed using Python Kivy. Geek Notes is a application that provides necessary resources for VTU engineering students, who are facing problem with Notes, Questiion Paper, Time Table, LAB Manuals. Tested On: Android such as Realme, Samsung, OnePlus, Redmi etc. Requirements: Kivy, Kivymd You can install kivy and kivymd by using pycharm terminal: "pip install Kivy", "pip install kivymd".
